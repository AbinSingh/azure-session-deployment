# 1. Is TCP part of OS stack?
Yes. TCP is typically implemented as part of the operating system's networking stack (the kernel). That's one of the most important concepts in networking.

# 2. Who implements TCP?

On Linux:
The Linux kernel implements TCP.

On Windows:
The Windows networking stack implements TCP.
