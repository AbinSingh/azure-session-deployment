                    USER
                      │
                      ▼
                 Browser (Chrome)
                      │
            POST /login HTTP Request
                      │
                      ▼
               localhost:8000
                      │
                      ▼
                 OS Socket
                      │
                      ▼
                  Uvicorn
                      │
          Parses HTTP into ASGI Scope
                      │
                      ▼
                 FastAPI App
                      │
            Session Middleware
                      │
      Reads/Writes Session Dictionary
                      │
                      ▼
               Route Handler
                      │
                HTTP Response
                      │
          Set-Cookie: session_id=...
                      │
                      ▼
                 Browser stores cookie


| Component     | Responsibility                    |
| ------------- | --------------------------------- |
| Browser       | Sends requests, stores cookies    |
| OS            | Owns TCP socket                   |
| Uvicorn       | HTTP server, converts HTTP → ASGI |
| FastAPI       | Routing                           |
| Middleware    | Loads/saves session               |
| Session Store | Remembers users                   |
| Route         | Business logic                    |
